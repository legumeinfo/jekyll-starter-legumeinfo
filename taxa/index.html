--- layout: taxa title: Taxa ---

This is the taxa page. It lists all of the taxa in your site. The theme does not provide a taxa page since not every site may want such a page, but a taxa layout is provided so sites that want a taxa page don't necessarily have to implement one from scratch.

The taxa listed on this page are loaded from a YAML file. The theme also generates a page for each species using the Jekyll datapage-generator plugin.
