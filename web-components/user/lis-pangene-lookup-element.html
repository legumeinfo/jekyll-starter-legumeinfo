<lis-pangene-lookup-element>

The <lis-pangene-lookup-element> provides a form for performing pangene lookups and displays the results. The lookup is performed using an external function provided to the component when it is added to the page. In this example, the component queries the {{ site.graphql_link }} using JavaScript provided by the {{ site.theme_link }}. The search form's genus, species, strain, assembly, and annotation selectors are also populated with data from the {{ site.graphql_name }}. See the source code for details.

Optionally, all lookups can be limited to a specific genus by setting the genus attribute/property of the <lis-pangene-lookup-element> component. This will cause the genus field of the lookup form to be automatically set and disabled so that users cannot change it. Similarly, all lookups can be limited to a specific species, assembly, and annotation by setting the species, assembly, and annotation attributes/properties, respectively. You can try setting these properties in this example using the pangeneLookupElement variable in the Web console. Additionally, the example text for the Genes form field can be provided by setting the genesExample attribute/property.
