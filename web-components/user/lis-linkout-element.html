<lis-linkout-element>

The <lis-linkout-element> component provides a generic interface for loading and displaying hyperlinks. In this example, links are loaded for a legume gene from the {{ site.graphql_link }} using JavaScript provided by the {{ site.theme_link }}. A new query can be issued using the linkoutElement.getLinkouts(...) method in the Web console.
