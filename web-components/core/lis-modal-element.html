<lis-modal-element>

The <lis-modal-element> component provides a generic modal. Specifically, the component is a wrapper for the UIkit modal and can be interacted with using the UIkit API.

In this example, the modal is populated with the <lis-simple-table-element> component. See the source code and documentation for details.
