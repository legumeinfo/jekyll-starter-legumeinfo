Web Components & GraphQL

The {{ site.theme_link }} supplies a copy of the {{ site.webcomponents_link }} and provides JavaScript for loading data from the {{ site.graphql_link }}. When combined, these technologies allow data to be dynamically loaded and visualized within an otherwise static Jekyll site.

This page is a demonstration of ALL of the Web Components provided by the {{ site.webcomponents_name }} and shows how to load data into the components from the {{ site.graphql_name }} using the JavaScript provided by the {{ site.theme_name }}.

Below are examples of ALL of the components in the {{ site.webcomponents_name }}. User Components are components intended for use by end users and provide rich functionality. Core Components are primitive components that are used to build and enhance user components. These should not be used as stand-alone user-facing components. Examples demonstrate how multiple components can be used together to create sophisticated functionality.

User Components:

  • <lis-gene-search-element>
  • <lis-linkout-element>
  • <lis-pangene-lookup-element>
  • <lis-publication-search-element>
  • <lis-qtl-search-element>
  • <lis-trait-association-search-element>
  • <lis-trait-search-element>

Core Components:

  • <lis-form-wrapper-element>
  • <lis-modal-element>
  • <lis-pagination-element>
  • <lis-simple-table-element>

Examples:

  • Using <lis-gene-search-element> with <lis-linkout-element> and <lis-modal-element>
